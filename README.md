## Template example for creating npm packages using vite

### Folder structure 
```bash
📂package-name
 ├── 📁src          
 │    ├── 📁lib
 │    └── 📄index.ts
 ├── 📄.gitignore
 ├── 📄package.json
 └── 📄vite.config.js
```

```bash
git clone https://github.com/andremalveira/vite-package-example.git package-name
```

```bash
npm i         # Instalation

npm run build # Package build
```
